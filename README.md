# Memoriza Expo

Versao do aplicativo Memoriza para abrir no VS Code e testar pelo Expo Go.

## Como rodar

1. Abra esta pasta no VS Code.
2. Abra o terminal do VS Code.
3. Rode:

```bash
npm install
npx expo start
```

4. Escaneie o QR Code com o app Expo Go no celular.
